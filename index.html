통합 결과 엑셀 자동 배치기

6개의 결과 파일과 1개의 전체 결과 템플릿을 넣으면, 지정한 위치에 자동으로 붙여 넣고 제목 / 방어폭 / 예상손절금도 함께 바꾼 뒤 새 엑셀 파일로 내려받는 도구입니다.
추가로 손절요약 파일 1개를 넣으면, 전체 / 손절 1·0개 / 손절 0개 블록도 자동으로 채웁니다.

1) 기본 값 입력

적용 위치: A2 / AC2 / AW2
필터링 블록은 자동으로 1차 필터링 {숏 제목}, 2차 필터링 {숏 제목} 형태로 들어갑니다.
적용 위치: A28 / AC28 / AW28
필터링 블록은 자동으로 1차 필터링 {롱 제목}, 2차 필터링 {롱 제목} 형태로 들어갑니다.

2) 파일 업로드

예: 전체 결과.xlsx
예: 손절요약_날짜정렬.xlsx
이 파일은 왼쪽 전체 / 가운데 손절 1,0개 / 오른쪽 손절 0개 블록을 자동으로 채웁니다.
6개 결과 파일은 각 파일의 첫 번째 시트에서 1~2행, A~L을 읽어 옵니다.
붙여넣기 위치는 네가 확인해준 대로 A~L / AC~AN / AW~BH 기준으로 처리합니다.
손절요약 파일은 왼쪽 블록에 A~H 전체, 가운데 블록에 손절횟수 1,0개 행만 A~D, 오른쪽 블록에 손절횟수 0개 행만 A~B를 넣도록 만들었습니다.

3) 붙여넣기 매핑

업로드 파일 읽는 범위 붙여넣는 위치
숏 전체결과1~2행 / A~LA3:L4
숏 1차 최적화1~2행 / A~LAC3:AN4
숏 2차 최적화1~2행 / A~LAW3:BH4
롱 전체결과1~2행 / A~LA29:L30
롱 1차 최적화1~2행 / A~LAC29:AN30
롱 2차 최적화1~2행 / A~LAW29:BH30
손절요약 파일전체 A~H / 필터된 A~D / 필터된 A~BO3:V끝 / AQ3:AT끝 / BJ3:BK끝
